# Shared Components

## Deck primitives

- `deck`
  - overall page shell
- `slide`
  - fixed-layout section container
- `metric-card`
  - headline stat and support meta
- `distribution-card`
  - donut chart shell plus legend list
- `target-panel`
  - PDP snapshot / hero listing panel
- `competitor-card`
  - competitor image and summary card
- `gallery-card`
  - image-heavy listing gallery tile
- `recommendation-card`
  - bullet recommendation block
- `plan-card`
  - closing-plan summary block

## Interaction notes

- The HTML deck is fixed-layout and print-friendly.
- “Editable” means the consuming project can swap sections or data while keeping the same primitives.
- Keep service-offering slides distinct from data-backed market slides.
